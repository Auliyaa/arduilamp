You'll need to print 2x **Shade_with_holes.STL** and 2x **Shade_with_mount.STL** for the sides. Skip the other files.
Instead use the **Base.stl** and **Top_cover.stl** provided in this repository for the top & bottom pieces.

Those were printed on a Creality Ender-3 using wood PLA.
